# Task Tracker Web Application

A simple and professional Task Tracker web application built using HTML, CSS, and JavaScript as part of a Full-Stack Internship assignment.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Task status displayed as Pending or Completed
- Persistent data using LocalStorage
- Clean and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Project Structure
- index.html
- style.css
- script.js

## How to Run the Project
1. Download or clone the repository
2. Open `index.html` in any modern web browser
